# GamemodeR

> 高级版灵魂出窍(切旁观, 切回生存传送回原位置)
>
> 可选的收费功能

感谢[方块君](https://github.com/Squaregentleman)的[gamemode](https://github.com/Squaregentleman/MCDR-plugins) 插件

## 使用

`!!spec` 旁观/生存切换

`!!tp <dimension> [position]` 传送至指定地点

`!!back` 返回上个地点

## 配置

### enable_free

默认值:`True`

是否免费

### spec_price

默认值:`10`

使用观察者模式每分钟的价格

### tp_price

默认值:`20`

每次传送的价格
